# ThingOrigin.js



## ThingOrigin.js是什么

ThingOrigin.js 是一款高性能 Web3D 引擎，为开发者提供从场景搭建到交互控制的全流程解决方案。

通过简洁高效的 API，可快速实现 3D 场景渲染、模型缓存，导入与控制、物理模拟、动画系统及数据可视化等功能，广泛适用于工业仿真、数字孪生等领域。

## 5秒轻松搭建场景

```javascript
let TO = new ThingOrigin('TOScene',document.getElementById('TO'));
```
